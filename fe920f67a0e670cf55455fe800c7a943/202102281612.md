# Interaction Layer
#architecture

> Model Behavior

Application (or Interaction) Logic is logic that decide what happen next. We can implement this with redux-thunk / react hooks. 

Example of this layer is:
- Auth layer 
- Logging
- Real-time subscriptions
- Complex rendering logic
- Metadata layer
- Validation
